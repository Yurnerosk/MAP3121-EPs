# ⚔️ Assignment

This assignment is divided into two main parts:

## Part 1 – Direct Problem (Heat Equation)

Implementation and analysis of numerical methods (explicit, implicit, and Crank-Nicolson) to solve the 1D heat equation with various initial conditions, boundary conditions, and source terms.

## Part 2 – Inverse Problem

Given the final temperature distribution, determine the intensities of point heat sources using least squares and QR factorization.
